# 筋トレログ

スマホアプリ「筋トレmemo」の代わりとして使える、ブラウザだけで動く筋トレ記録アプリです。
サーバー不要・ライブラリ不要の単一ページアプリで、データは端末のブラウザ（localStorage）に保存されます。

## 使い方

`kintore/index.html` をブラウザで開くだけで動きます。

- **記録**: 日付・部位・種目・セット（重量×回数）・メモを入力。「前セットをコピー」や、最近使った種目チップから前回のセット内容を呼び出せます。
- **履歴**: 日付ごとの一覧。種目・部位・メモで絞り込み、個別削除ができます。
- **分析**: トレーニング日数・総挙上量などのサマリー、種目別ベスト（PR・推定1RM）、種目ごとの推移グラフ（最大重量／ボリューム／推定1RM）、週ごとの総量グラフ。
- **データ**: CSV / JSON のエクスポートとインポート。

GitHub Pages を有効にすれば `https://<ユーザー名>.github.io/heart-salon/kintore/` としてスマホからも使えます。

## 筋トレmemoからのデータ移行

筋トレmemoにはエクスポート機能がないため、次の手順で移行します。

1. 筋トレmemoの履歴画面を日付ごとにスクリーンショットする
2. スクリーンショットをClaudeに渡して、下記のCSV形式に書き起こしてもらう
3. このアプリの「データ」タブ → インポートでCSVを取り込む

### CSV形式

1行 = 1セット。ヘッダー行は日本語（日付,種目,部位,セット,重量,回数,メモ）でも認識されます。

```csv
date,exercise,body_part,set_no,weight_kg,reps,memo
2026-07-01,ベンチプレス,胸,1,60,10,
2026-07-01,ベンチプレス,胸,2,60,8,
2026-07-01,ラットプルダウン,背中,1,50,12,順手ワイド
```

- 日付は `2026-07-01` / `2026/7/1` / `2026年7月1日` のいずれの形式でもOK
- 部位が空欄でも、種目名から自動で推定します
- 同じデータを2回取り込んでも、完全一致する記録は重複スキップされます

## データ仕様（分析用）

localStorage キー: `kintore-log-v1`

```json
[
  {
    "id": "一意ID",
    "date": "YYYY-MM-DD",
    "exercise": "ベンチプレス",
    "bodyPart": "胸",
    "sets": [{ "weight": 60, "reps": 10 }],
    "memo": ""
  }
]
```

エクスポートしたCSV/JSONをそのままClaudeに渡せば、停滞期の検出やメニュー提案などの追加分析を依頼できます。
